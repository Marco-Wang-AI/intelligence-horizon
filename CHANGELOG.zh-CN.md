# 更新日志

这里记录 AGI Radar / AGI 小雷达每个版本的重要变化。

[English changelog](./CHANGELOG.md)

## [尚未发布]

#### 新增

- 接入 Cloudflare Web Analytics 统计代码，用于查看线上站点访问量和访问次数。
- 新增两条有来源的 ASI 雷达信号：伊利亚·苏茨克维创办 Safe Superintelligence，以及利奥波德·阿申布伦纳的《Situational Awareness》叙事。
- 增加轻量的信号提交卡片，在 AGI / ASI 雷达记录数量不均时平衡版式。
- 更新德米斯·哈萨比斯头像，加入 DeepMind、诺贝尔奖、AlphaFold、AlphaGo 和神经网络相关视觉元素。
- 将德米斯·哈萨比斯的雷达标签更新为“把游戏做成智能引擎的人”，并在 AGI 雷达扫描中展示梁文锋信号。
- 将梁文锋的雷达标签更新为“开源节流派”。
- 更新罗福莉头像，加入淡化的 DeepSeek 与小米 MiMo 推理元素，并将雷达标签更新为“MiMo 推理引擎手”。
- 将张鹏的雷达标签更新为“清华 GLM 长跑者”。
- 将 OpenAI 的 AGI 信心信号改为山姆·奥特曼个人归属，并将雷达标签更新为“把 ChatGPT 推向世界的操盘手”。
- 更新山姆·奥特曼头像，加入 OpenAI / ChatGPT、YC 和全球扩散相关视觉元素。
- 更新梁文锋头像，加入 DeepSeek、开源节流和 AGI 目标相关视觉元素。
- 为雷达记录头像增加大图预览弹窗，支持复制图片链接和下载，同时卡片其他区域仍跳转公开来源。
- 中文界面的雷达人物姓名改为中文显示，避免中文标题和人物卡片里混用英文名。
- 修正杨立昆中文名在雷达标题中的错字，并将卡片来源标签统一为“来源”。

## [1.1.0] - 2026-06-07

#### 调整

- 移动端打开后直接展示雷达，不再加载较重的封面背景图。
- 移动端将双周更新信息收进 AGI 雷达卡片。
- 雷达事件点由字母缩写升级为已有的人物与产品漫画头像。
- 移动端雷达扫描会自动聚焦不同信号，并显示对应距离。
- 不再使用倒计时钟式的“加减天数”，改成雷达信号“靠近或远离多少米”。
- 移除遗留的“快了区”，明确雷达中心代表当前预测位置。
- 将“动态 / Updates”统一改为“雷达记录 / Radar Log”。
- 重写开源区域，用更平和的方式邀请大家贡献信号、来源、文案和想法。
- 精简公众问卷：只保留四道必答选择题，人口信息和开放问题全部改为选填。
- 修正窄屏下语言切换按钮换行的问题。
- 移动端专家预测与大众感受保持左右并排。
- AGI / ASI 雷达卡和定义卡改为可左右滑动。
- 在每张雷达卡的预测对比下增加快捷投票入口。
- 补充 AGI / ASI 中文释义，并说明中心年份代表当前专家预测。
- 按证据强度重新校准信号距离，在 OpenClaw 旁标明创始人彼得·斯坦伯格，并将正式更名时间细化到 2026 年 1 月。
- 微调移动端扫描节奏，让距离提示在扫描线真正到达头像后再出现。
- 将“小雷达当前基准”改为“小雷达当前定义”，并说明高票候选会在双周更新时进入评审。
- 移动端雷达记录按 AGI / ASI 分成两栏，可左右滑动切换。
- 删除内部的 Method Desk 占位项，让公开记录只保留有来源的人物、产品和观点。
- 修正窄屏下左侧雷达事件距离标签被圆盘裁切的问题。
- 删除雷达记录分栏标题中重复出现的 AGI / ASI 字样。

#### 新增

- 雷达记录卡片和当前选中的雷达事件可以直接跳转到公开来源。
- 增加“雷达米”的说明：它是项目内部的相对距离，不是物理测量。
- 增加直接前往 GitHub 提交雷达信号的入口。

#### 社区贡献

- 移动端雷达优先、头像信号、距离语言、开源文案和导航命名，由项目发起人的妻子（一位 AI 产品经理）提出。
- 雷达记录增加来源跳转，由一位通过微信联系项目的早期读者提出。
- 问卷精简方向，由项目发起人根据首发体验提出。

## [1.0.0] - 2026-06-06

#### 新增

- 中英双语 AGI / ASI 雷达，同时展示专家预测和大众感受。
- 带漫画头像的专家与产品信号记录。
- 可点击出处的 AGI / ASI 候选定义。
- 科幻作品 AI 时间轴。
- 中文问卷星与英文 Google Forms 投票入口。
- 双周更新节奏和开源贡献模板。
