# 更新日志

这里记录 AGI Radar / AGI 小雷达每个版本的重要变化。

[English changelog](./CHANGELOG.md)

## [尚未发布]

### 计划纳入 v1.1.0

#### 调整

- 移动端打开后直接展示雷达，不再加载较重的封面背景图。
- 雷达事件点由字母缩写升级为已有的人物与产品漫画头像。
- 不再使用倒计时钟式的“加减天数”，改成雷达信号“靠近或远离多少米”。
- 将“动态 / Updates”统一改为“雷达记录 / Radar Log”。
- 重写开源区域，用更平和的方式邀请大家贡献信号、来源、文案和想法。
- 精简公众问卷：只保留四道必答选择题，人口信息和开放问题全部改为选填。

#### 新增

- 雷达记录卡片和当前选中的雷达事件可以直接跳转到公开来源。
- 增加“雷达米”的说明：它是项目内部的相对距离，不是物理测量。
- 增加直接前往 GitHub 提交雷达信号的入口。

#### 社区贡献

- 移动端雷达优先、头像信号、距离语言、开源文案和导航命名，由项目发起人的妻子（一位 AI 产品经理）提出。
- 雷达记录增加来源跳转，由一位通过微信联系项目的早期读者提出。
- 问卷精简方向，由项目发起人根据首发体验提出。

## [1.0.0] - 2026-06-06

#### 新增

- 中英双语 AGI / ASI 雷达，同时展示专家预测和大众感受。
- 带漫画头像的专家与产品信号记录。
- 可点击出处的 AGI / ASI 候选定义。
- 科幻作品 AI 时间轴。
- 中文问卷星与英文 Google Forms 投票入口。
- 双周更新节奏和开源贡献模板。
