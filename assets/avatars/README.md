# Avatar Assets

Illustrated radar-log avatars for AGI Radar / AGI 小雷达.

These are project-ready square thumbnail assets used by the public site. The avatar preview lightbox, copy-link action, and download action should use the matching full-resolution files in `assets/avatar-originals/`.

Source/reference photos are kept locally under `local/avatar-references` or `local/avatar-sources` and are intentionally not committed.
