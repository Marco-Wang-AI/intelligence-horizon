# v1.1.0：这次真的更像雷达了

这个版本让核心比喻变得更统一：不同信号会让 AGI / ASI 靠近或远离雷达中心，并用轻松的相对“雷达米”表示距离变化。

## 主要更新

- **移动端雷达优先。** 打开网站后先看到 AGI 雷达，不再等待大幅封面图加载。
- **移动信息重新排布。** 双周更新收进 AGI 卡片，专家预测与大众感受左右并排。
- **左右滑动。** AGI / ASI 雷达和两张定义卡都可横向切换，并露出下一张卡片作为提示。
- **头像代替缩写。** 雷达信号点开始使用项目已有的人物与产品漫画头像。
- **扫描会选中信号。** 雷达扫过头像时，会自动显示对应事件和靠近 / 远离距离。
- **米代替天。** 每条事件显示它让雷达靠近或远离了多少。
- **更容易看懂。** 卡片补充 AGI / ASI 中文释义，并解释大字年份是当前专家预测。
- **重新校准信号。** 距离更统一地按证据强度判断；OpenClaw 同时标明创始人 Peter Steinberger，并细化到 2026 年 1 月正式更名。
- **更快参与。** 每张雷达卡的预测对比下都增加了投票入口。
- **出处一步直达。** 点击雷达记录或当前事件，可直接打开对应的公开来源。
- **雷达记录更短。** 移动端可在 AGI / ASI 两栏之间横滑，并移除了内部 Method Desk 占位项。
- **定义也会更新。** 高票候选定义会随双周节奏进入下一期评审。
- **问卷变短。** 四道核心选择题必答，其余问题全部选填。
- **开源入口更友好。** 可以直接前往 GitHub 查看项目或提交一条新信号。
- **最后一轮移动端打磨。** 左侧距离标签不再被雷达裁切，雷达记录标题也不再重复显示 AGI / ASI。

## 说明

- “雷达米”是项目内部的相对表达方式，不是科学或物理测量。
- 大众预测仍通过中文和英文问卷收集，并按照双周节奏更新到网站。
